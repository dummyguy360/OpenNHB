# OpenNHB
A Decompilation of Noise's Halloween Bash v1.0.0

GameMaker Version Used: ```2023.11.1.129```

Packages Imported
- GMEXT-FMOD v1.0.4
- NekoPresence
- GameFrame
- Input v8.0.3
- DotObj v5.5.0
- Easings

### NOTES
- A FMOD project recreation is included, but is currently unfinished and has some minor issues
